# RPG-DPS-Calculator
1. Програма Universal RPG DPS Calculator призначена для геймерів та любителів рольових ігор (RPG). 
Вона дозволяє автоматизувати розрахунок ефективності ігрового спорядження, вираховуючи показник шкоди за секунду (DPS — Damage Per Second). 
Це допомагає гравцям обирати найкращу комбінацію зброї та характеристик персонажа без необхідності проводити складні обчислення вручну.

2. Системні вимоги Для коректної роботи програми необхідні:

    Операційна система: Windows 10 або Windows 11 (64-bit).

    Простір на диску: 50 МБ.

    Оперативна пам'ять: 2 ГБ або більше.

    Клавіатура та миша.

3. Встановлення та запуск Програма не потребує складного встановлення, оскільки розповсюджується у вигляді портативного файлу (.exe).

    Завантажте файл main.exe з розділу "Releases" на офіційній сторінці репозиторію GitHub.

    Збережіть файл у будь-яку зручну папку на комп'ютері.

    Двічі клікніть на файл main.exe для запуску.

4. Інтерфейс програми Головне вікно програми складається з двох основних блоків:

    Характеристики персонажа (Зверху): Поля для введення сили героя та параметрів критичного удару.

    Параметри зброї (Знизу): Поля для введення базової шкоди зброї та швидкості атаки.

    Кнопка дії: "Розрахувати DPS".

    Зона результату: Відображення фінального числа DPS.

5. Інструкція з використання Щоб виконати розрахунок, виконайте наступні кроки:

    Введіть силу персонажа: У поле "Сила персонажа" введіть числове значення (наприклад, 100).

    Налаштуйте критичний удар:

        У поле "Шанс криту (%)" введіть ймовірність (наприклад, 20 для 20%).

        У поле "Множник криту (x)" введіть коефіцієнт шкоди (наприклад, 1.5 для 150%).

    Введіть параметри зброї:

        У поле "Шкода зброї" введіть атаку, вказану на предметі (наприклад, 50).

        У поле "Швидкість атаки" введіть кількість ударів за секунду (наприклад, 1.2).

    Отримайте результат: Натисніть зелену кнопку "Розрахувати DPS".

    Результат з'явиться внизу вікна жирним шрифтом (наприклад, DPS: 250.0).

6. Вирішення проблем (Troubleshooting)

Якщо програма видає вікно "Помилка", перевірте чи не ввели ви літери чи спецсимволи. Використовуйте тільки цифри та крапку для дробів.

Якщо програма не запускається, можливо її блокує антивірус, додайте програму у виключення або запустіть від імені адміністратора.

Якщо результат дорівнює 0, перевірте чи не ввели ви 0 в швидкості атаки або шкоді.

7. Технічна підтримка У разі виявлення багів, будь ласка, створіть "Issue" у репозиторії проекту на GitHub із описом проблеми та скріншотами.
